Place imported Unreal Texture2D target assets for image evolution in this folder.

Recommended workflow:
1) Copy raw source image files to Resources/ReferenceImages for archival if desired.
2) Import image files into Content/Targets in Unreal Editor.
3) Assign the imported Texture2D asset to TargetTexture on BP_ImageEvolution_Main.

This folder is tracked by source control through plugin .gitignore exceptions.
